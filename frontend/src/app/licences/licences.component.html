Opensource-Lizenzen

Die Working together-Anwendung steht unter der GPL V3. Es steht Dir (im Rahmen der GPLv3) frei die komplette Anwendung zu verwenden, zu kopieren oder an Deine Anforderungen anzupassen.

Verwendete Opensource-Software

Datenbank

Als Datenbank wird MariaDB verwendet. MariaDB steht unter der GPL V2.

Server

Für die Backend-Server-Software wird Spring-boot verwendet. Spring-boot steht unter der Apache Lizenz V2

Frontend

Für die Frontend-Software wird Angular verwendet. Angular steht unter der MIT-Lizenz

Kalender

Als Kalender-Komponente wird DayPilot-lite verwendet. DayPilot-lite steht unter der Apache Lizenz V2
